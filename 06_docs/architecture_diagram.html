Each layer is implemented as a runnable Jupyter notebook (01–10). Outputs flow sequentially: raw data → features → PD → staging → financial intelligence → decision → stress → monitoring.
The RAROC Gate (Layer 5) is the primary policy lever. Changing the RAROC threshold does not require retraining any model — it is a post-score policy calibration step.
All numbers derive from 307,511 Home Credit applications (training set) and 61,503 loans in the decision engine evaluation set. LGD = 0.45 base case, cost of capital = 17%, Basel risk weights throughout.
