Step 3 — Multicollinearity (VIF)

Variance Inflation Factor threshold: VIF < 5. Correlated pairs resolved by retaining higher-IV feature. 12 features removed at this stage.

Step 4 — WoE Encoding

Each feature binned into 5–10 monotone bins. Weight of Evidence transformation applied:

WoE_i = ln(Events_i / Non-Events_i) × (Total Non-Events / Total Events) IV = Σ (Events_i% - Non-Events_i%) × WoE_i

Monotone binning enforced for regulatory interpretability. Non-monotone bins flagged for review.

2. PD Model Development

The primary PD model uses logistic regression on WoE-encoded features. A secondary XGBoost model is trained for challenger comparison and SHAP explainability. The logistic regression champion is used for scorecard construction and regulatory submission; XGBoost is used for feature importance validation.

PD = σ(β₀ + β₁·WoE(EXT_SOURCE_2) + β₂·WoE(DAYS_BIRTH) + β₃·WoE(DAYS_EMPLOYED) + β₄·WoE(AMT_CREDIT_INCOME_PCT) + β₅·WoE(PAYMENT_RATE) + ... + β₂₄·WoE(feature_24)) where σ(x) = 1 / (1 + exp(-x))

4. Calibration

A well-discriminating model is not necessarily well-calibrated. Calibration measures whether predicted PD = actual default rate. For IFRS 9 provisioning, ECL = PD × LGD × EAD — a systematically biased PD directly inflates or deflates the provision.

Band-Level PD Ratio (Actual DR / Model PD)

PD Ratio < 1.0 = model overpredicts default (conservative). PD Ratio > 1.0 = model underpredicts default (optimistic). Acceptable range: 0.7× – 1.3×.

Consistent over-prediction: The model predicts default at 4–8× the actual observed rate across all bands. This is structurally conservative — suitable for regulatory ECL provisioning where over-statement of risk is preferable. However, for RAROC computation, over-stated PD inflates EL and deflates NI, potentially causing economically sound loans to appear marginally negative. The RAROC gate threshold (≥ 2.5%) partially accounts for this by filtering out only those loans where RAROC is substantially negative, not just marginally so.

Recalibration schedule: Annual recalibration using Platt scaling on the most recent 12-month observed outcomes. If PD Ratio drifts above 1.3× or below 0.5× at the portfolio level, trigger off-cycle recalibration. PSI > 0.25 on score distribution also triggers recalibration review. 5. Scorecard Construction

The logistic regression output is converted into an integer credit scorecard using the standard PDO (Points to Double the Odds) framework for underwriter interpretability.

Score = Base_Score + Σ (WoE_i × Coefficient_i × Factor) Factor = PDO / ln(2) Offset = Base_Score - Factor × ln(Base_Odds) PDO = 20 points | Base_Score = 600 | Base_Odds = 1:10 (Default:Non-Default)

6. Population Stability Index (PSI)

PSI measures the shift in score distribution between a reference population (training time) and a monitoring population (deployment time). A shift signals that the model's input population has changed, potentially invalidating the model's predictions.

PSI = Σ (Actual_i% - Expected_i%) × ln(Actual_i% / Expected_i%) Threshold: PSI < 0.10 → Stable (no action) PSI 0.10–0.25 → Amber (investigate) PSI > 0.25 → Red (recalibrate or rebuild)

CSI — Characteristic Stability Index

CSI applies the PSI formula to individual input features rather than the score output. A feature with CSI > 0.25 suggests the feature's distribution has shifted significantly — it may no longer be predictive in the same way as at training time.

Monitoring frequency: PSI and band-share — monthly. Portfolio RAROC and default rate — monthly. Vintage cohort review — quarterly (at 3M, 6M, 12M, 24M loan age checkpoints). IFRS 9 stage migration — quarterly (aligned with IFRS 9 reporting). ECL provision adequacy — semi-annual or at each stress event.

Escalation protocol: Amber triggers require Credit Risk team investigation within 5 business days and report to CRO within 10 days. Red triggers require immediate CRO notification, credit committee review within 15 days, and potential model freeze pending investigation. Model freeze means no new approvals above current gate until root cause is identified.
